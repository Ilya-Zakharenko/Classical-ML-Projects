# <center> **PROJECT: MolPredict. Exploring Chemistry.**


Этот проект представляет собой комплексное исследование в области химии с использованием машинного обучения. Проект включает EDA, задачи регрессии и классификации на основе молекулярных данных.

---


### **Описание**

Проект **MolPredict: Exploring Chemistry** направлен на анализ и моделирование данных, связанных с молекулярными свойствами. Содержит три основных раздела:  

- **EDA (Исследовательский анализ данных)**: Анализ данных для понимания их структуры и закономерностей. 

- **Регрессия**: Построение моделей для предсказания количественных характеристик молекул:  
  - `IC50` (ингибирующая концентрация $50$%).  
  - `CC50` (цитотоксическая концентрация $50$%).  
  - `SI` (селективный индекс).  

- **Классификация**: Построение моделей для бинарной классификации:  
  - `IC50` > медианы.  
  - `CC50` > медианы.  
  - `SI` > медианы.  
  - `SI` > $8$.

Проект может быть полезен для фармацевтических исследований, разработки лекарств и анализа химических соединений.

---


### **Структура проекта**

- `eda/`: Исследовательский анализ данных (**EDA**),
- `regression/`: Модели регрессии для `IC50`, `CC50` и `SI`,  
- `classification/`: Модели классификации для указанных условий.

---


### **Используемые инструменты**
- `Python`
- `Pandas`, `NumPy` (для обработки данных),
- `Scikit-learn` (для регрессии и классификации),
- `Matplotlib`, `Seaborn`, `Plotly` (для визуализации).
